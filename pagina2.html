3. Legenda — Como ler os wireframes

Estrutura de cada página

• Topo (10%): Título + barra de filtros
• KPIs (20%): 4-5 cards com números-chave
• Área principal (60%): 2-4 visuais
• Rodapé (10%): Tabelas detalhadas

Texto cinza abaixo dos visuais

Indica exatamente quais campos e medidas arrastar:
• X/Y: campo do eixo
• Valor: medida DAX a usar
• Visual: tipo de gráfico do Power BI

Filtros por página

• P1: Período + Front + Produto + Moeda
• P2: Período + Produto tentado
• P3: Período + Front + Campanha
• P4: Período + Produto + Moeda
• P5: Período + Plataforma + Produto
